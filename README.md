# STM32_Embedded_System

This repository showcases a series of bare-metal STM32 embedded systems projects developed using CMSIS and PlatformIO, covering topics such as GPIO control, timers, interrupts, ADCs, PWM, UART communication, sensor integration, motor control, and robotics. Projects range from LED sequencing and seven-segment display controllers to ultrasonic radar systems, servo feedback control, and an autonomous infrared sensor suite robot. These projects strengthened my experience with real-time embedded systems, low-level firmware development, and hardware-software integration. 

Developed in collaboration with Ryan Atkins, a fellow Electrical Engineering student at the University of Delaware, Class of 2028.
